# 🛠️ Local Service Provider

A full-stack web application that connects users with nearby service providers like electricians, plumbers, drivers, and more. Users can search, filter, and choose providers based on distance, budget, and service type.

---

## 🚀 Features

✨ User Signup & Login (Authentication)
🔍 Search service providers by category
📍 Filter providers by distance & budget
⭐ View ratings, reviews, pricing & ETA
🧑‍🔧 Multiple services (Electrician, Plumber, Driver, etc.)
🌱 Pre-seeded database with 40+ providers

---

## 🛠️ Tech Stack

* ⚙️ Backend: Node.js, Express.js
* 🗄️ Database: MongoDB (Mongoose)
* 🔐 Authentication: bcrypt
* 🌐 REST API with filtering & querying
* 🎨 Frontend: HTML, CSS, JS

---

## ⚙️ Installation

```bash
git clone https://github.com/sourabh1021/local-service-provider.git
cd local-service-provider
npm install
node server.js
```

---

## 🔗 API Endpoints

* **POST** `/api/signup` → Register user
* **POST** `/api/login` → Login user
* **GET** `/api/providers` → Get providers with filters

---

## 📊 Key Functionalities

* Dynamic provider filtering using query parameters
* Secure password hashing
* Location-based mock data (lat/lng)
* Scalable REST API design
